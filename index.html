Cette page permet de explorer et faire des analyses simples des chaînes de référence dans la ressource Democrat RC 19-21. Cette ressource a été dévéloppée dans le cadre de la thèse de Silvia Federzoni. La ressource est constituée de toutes les chaînes de référence d'au moins 3 maillons annotées dans le corpus Democrat.
La ressource Democrat RC 19-21 est mise à disposition sur la plateforme Ortolang : Democrat RC 19-21.
