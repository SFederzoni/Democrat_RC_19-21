Cette page pormet de explorer et faire des analyses simples des chaînes de référence dans la ressource Democrat RC 19-21. Cette ressource a été dévéloppée dans le cadre de la thèse de Silvia Federzoni. La ressource est constituée de toutes les chaînes de référence d'au moins 3 maillons annotées dans le corpus Democrat.

Dans cette page il est possible d'intérroger la ressource pour faire des analyses des CR qui y sont contenues. L'onglet "statistiques" permet de visualiser les résultats obtenus dans le cadre de la thèse de Silvia Federzoni.
